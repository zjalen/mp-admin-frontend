# mp-admin 前端  UI

基于 `Vue` & `Vuetify`, 如需自定义使用建议 `fork` mp-admin 项目后在本地打包上传使用。

默认使用 `vue-cli3` 配置。

## 安装组件
```
yarn install
```

### 本地开发测试

默认配置了代理接口，指向 `http://localhost:8000`
```
yarn serve
```

### 打包方式

默认修改了打包路径，打包文件将存放于上级目录的 `resources` 路径
```
yarn build
```

### 严格代码格式检测
```
yarn lint
```
